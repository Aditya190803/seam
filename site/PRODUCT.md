# Seam Product Context

## Product Purpose

Seam is a local-first CLI and MCP server that gives coding agents semantic access to repositories without re-reading the whole codebase on every task. It indexes your code once, then lets AI agents query it instantly with semantic search, drastically improving agent performance and context accuracy.

## Users

**Primary**: Individual developers using AI coding agents (Claude, Codex, Cursor, etc.) who want faster, more accurate agent responses without repeatedly feeding entire codebases into context windows.

**Secondary**: 
- AI agent builders who need a semantic code search backend
- Development teams standardizing their agent tooling
- Power users optimizing local AI workflows

## Brand Principles

- **Local-first, always**: Privacy and offline operation are not negotiable. No data leaves the machine.
- **Zero-config by default**: Should work instantly with sensible defaults. Power users can configure.
- **Fast and invisible**: Indexing happens once. Queries feel instant. Agents don't wait.
- **Trust through transparency**: Open source, inspect the code, own your index.

## Tone

Technical but not opaque. Direct, confident, efficient. Speaks to developers who value performance and simplicity. Not playful, not corporate. Factual authority.

## Anti-references

- Generic SaaS landing pages with stock photos and fluffy copy
- Dark blue gradient "AI tool" aesthetic
- Over-explained basics (don't patronize developers)
- Hacker Terminal™ neon green on black clichés
- Feature comparison tables that bash competitors
